6lbr

A deployment-ready 6LoWPAN Border Router solution based on Contiki

Download this project as a .zip file Download this project as a tar.gz file

A 6LoWPAN Border Router connects your 6LoWPAN devices to the Internet and is responsible for handling traffic to and from the IPv6 and 802.15.4 interfaces. 6LBR is a deployment-ready solution for border routers, based on Contiki OS.

Getting Started

  • The 6LBR wiki documentation provdes an overview of the project and gives some insight on the implementation.
  • If you already own one of the supported platforms, grab a couple of your favorite Contiki motes and get started right away!

Contact

The 6LBR issue tracker is open to all. Also, feel free to contact us for feedback or bugfixes at 6lbr@cetic.be
